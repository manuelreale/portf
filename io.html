DESCRIPTION
IO is a mobile application that acts as a single access point allowing users to interact with local and national Italian public services. This project was dedicated to assess and enhance the usability of the IO mobile application through a redesign.

APP ANALYSIS
First the team conducted a detailed analysis of the current app, reconstructing the current Information Architecture, the main features, the main tasks and the possible mental model behind each of them. Then the team conducted an Heuristic Evaluation of the application and evaluated the usability of the app with user tests collecting time and number of steps to complete each task as well as using SUS scores and Premo tests. At the end of the analysis phase the team found that the main problems of the app were connected to: Information Architecture, Navigation, Affordances, Terminology and Visual language.

CARD SORTING AND TREE TESTING AND INFORMATION ARCHITECTURE
The first step to redesign the app was to reorganize the information architecture, the team performed open and then closed card sorting session and later used tree testings to validate the new organization, as a result a new IA was built.

ITERATIVE DESIGN
To redesign the interface of the app we followed an iterative process, first we tested and iterated two wireframes and then we performed an A/B testing of a first UI draft, then the team created a design system and finally designed the final UI Prototype suing Figma.
